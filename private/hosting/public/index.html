AI Agent Management Dashboard

Deploy AI agent teams from a tactical command center. Multi-AI routing, real-time monitoring, pixel art virtual office. Everything runs locally — your data never leaves your machine.

Windows One-Click Install
PS> irm https://raw.githubusercontent.com/cwen0708/aegis/refs/heads/main/scripts/install-windows.ps1 | iex

Open PowerShell as Administrator and paste the command above.
Or download the setup-aegis.bat installer and double-click to run.
Source code: github.com/cwen0708/aegis

// capabilities

Built for AI-Native Workflows

Not just another kanban — Aegis is purpose-built for orchestrating AI agents across planning, coding, and verification stages.

🎯
Multi-AI Routing
Route tasks to the right AI: Gemini for planning, Claude for coding. Per-stage member assignment with automatic provider detection.
🏢
Virtual Office
Pixel art office with AI characters walking between desks. Click any agent for AVG-style dialog with anime portraits.
📋
Kanban Automation
Cards flow through Backlog → Planning → Developing → Verifying → Done. AI picks up pending cards automatically.
📡
Real-time Monitoring
WebSocket-powered dashboard showing CPU/RAM, running tasks, live log streaming, and token cost analytics.
🧠
Agent Memory
Per-member short-term and long-term memory. Agents remember past tasks, patterns, and project-specific conventions.
🔒
Local-First
SQLite database, local file system. No cloud dependency. Carry the entire system on a USB drive.
