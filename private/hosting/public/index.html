Setup

AI 開發團隊的調度中心。一鍵安裝，自動配置 Python、Node.js、AI CLI， 雙擊桌面捷徑即可啟動你的 AI 工作平台。

Windows Linux macOS soon GitHub
Download Installer

setup-aegis.bat — Windows 10 / 11

或者，在終端機中執行：
powershell -c "irm https://raw.githubusercontent.com/cwen0708/aegis/main/scripts/install.ps1 | iex" click to copy
wget -O install.ps1 https://raw.githubusercontent.com/cwen0708/aegis/main/scripts/install.ps1 && powershell -ExecutionPolicy Bypass -File install.ps1 click to copy
在終端機中執行：
curl -fsSL https://raw.githubusercontent.com/cwen0708/aegis/main/scripts/install.sh | bash click to copy
wget -qO- https://raw.githubusercontent.com/cwen0708/aegis/main/scripts/install.sh | bash click to copy

支援 Debian / Ubuntu / Fedora / CentOS

01
下載
點擊上方按鈕，下載 setup-aegis.bat 安裝腳本
02
執行
雙擊執行，自動安裝 Git、Python、Node.js 及所有依賴
03
啟動
雙擊桌面「Aegis」捷徑，瀏覽器自動開啟
01
